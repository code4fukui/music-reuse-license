# Music Reuse License Template

[English](README.md) | [日本語](README.ja.md)

This is a bilingual license template that permits the reuse of music, sound recordings, lyrics, artwork, metadata, and similar materials while prohibiting their use for AI or machine-learning training and development.

## Usage

1. Use this repository as a template, or place `LICENSE` in the repository for your work.
2. Replace all of the following variables in `LICENSE`.

| Variable | Description | Example |
| --- | --- | --- |
| `{{YEAR}}` | Year of publication or the year used in the copyright notice | `2026` |
| `{{COPYRIGHT_HOLDER}}` | Name of the copyright holder | `Taro Yamada` |

Example after replacement:

```text
Copyright (c) 2026 Taro Yamada
```

As needed, include attribution information such as the work title, author name, and distribution source in the work's README or metadata.

## Terms

- Permits reproduction, distribution, performance, and modification for commercial and non-commercial purposes
- Requires the license notice, copyright notice, attribution, and an indication of modifications
- Prohibits use for AI or machine-learning training, evaluation, development, improvement, and related purposes
- Includes a governing English version and a Japanese translation for reference

## Using the Template

You may use the text of `LICENSE` after replacing the copyright holder's name and year. Once applied to a work, keep the file with that work as its licensing terms.
