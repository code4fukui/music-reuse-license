# Music Reuse License Template

[English](README.md) | [日本語](README.ja.md)

音楽、録音物、歌詞、アートワーク、メタデータなどの再利用を許可しつつ、AI・機械学習での学習や開発への利用を禁止するための、英日併記ライセンス雛形です。

## 使い方

1. このリポジトリをテンプレートとして使用するか、`LICENSE` を対象作品のリポジトリへ配置します。
2. `LICENSE` 内の次の変数をすべて置き換えます。

| 変数 | 内容 | 例 |
| --- | --- | --- |
| `{{YEAR}}` | 公開年または著作権表示に使用する年 | `2026` |
| `{{COPYRIGHT_HOLDER}}` | 著作権者名 | `Taro Yamada` |

置換後の例:

```text
Copyright (c) 2026 Taro Yamada
```

必要に応じて、作品名、作者名、配布元などの帰属情報も作品の README やメタデータに記載してください。

## 内容

- 商用・非商用での複製、配布、演奏、改変を許可
- ライセンス表示、著作権表示、変更表示を要求
- AI・機械学習の学習、評価、開発、改善などへの利用を禁止
- 英語版を正文、日本語版を参考訳として収録

## 注意

これは独自ライセンスの雛形であり、Creative Commons、オープンソースライセンス、Open Definition 準拠ライセンスではありません。利用目的や法域に適しているか判断が必要な場合は、法律の専門家に確認してください。

## 雛形自体の利用

`LICENSE` の本文は、著作権者名と年を置き換えて利用できます。作品へ適用した後は、その作品の利用条件として同ファイルを維持してください。
